Theming an Extension

You can easily theme an extension using the provided classes. We provide a light and dark theme mode, shown off in this page. To use light mode: 

e-theme-light
and dark: 
e-theme-dark
All other helper classes leverage this, so it's important to denote what theme your extension is in. This class can also be updated as needed, meaning you can easily update the theme as the user swaps between light and dark modes.

$black
$white
$brand
$brand-alt
$accent
$live
$alert
$success

$background-base-{theme}
$background-alt-{theme}
$background-alt-2-{theme}
$background-backdrop-{theme}
$text-base-{theme}
$text-alt-{theme}
